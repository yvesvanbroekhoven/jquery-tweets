jQuery tweets

The main idea behind this plugin was to be able to simple & flexible way to show tweets on your website.

To avoid hitting the rate limit, the results will be session cached client side.

I'm aware of the Twitter API change coming in March 2013, but we're working on a solution for that.

How to use

You can find the documentation on the Github page:
https://github.com/yvesvanbroekhoven/jquery-tweets
